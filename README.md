# B-tree indexing

### Prerequisites

- CMake

### Building the Project (for linux)


1. Create a build directory and navigate to it:
```sh
mkdir build
cd build
```

2. Run CMake to configure the project:
```sh
cmake ..
```

3. Build the project:
```sh
cmake --build .
```

### Running the Project

After building, you can run the executable located in the `build` directory. Example:

```bash
./Sbd2
```
